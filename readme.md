本プロジェクトは、STMicron社のマイコンテストボード
Nucleo F401RE向けのプロジェクトになります

マイコンボードのPA9およびPA10をUARTに設定しており、
本UART端子を用いてPCのターミナルソフトと通信をすることができます
通信設定は以下の通り
ボーレート：115200bps
bit長     ：8bit
パリティ  ：なし
Stop bit  ：1bit

コマンドは以下の通りとなります

pon [port]
指定されたポートにHiを出力します
poff [port]
指定されたポートにLoを出力します
pin [port]
指定されたポートのレベルを読みます

[port]には読み書きをしたいポートの名前を設定してください
例えば、PA01にHIを出力したい場合は
pon pa1となります
